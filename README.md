# SmartTodoApp

A simple Expo React Native todo app with a clean smart-style UI.

## Features
- Add todo tasks with title, detail, and priority
- Mark tasks complete and delete them
- Filter tasks by all / active / completed
- Smart summary with quick task counts and priority badges

## Run locally

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start Expo:
   ```bash
   npm run start
   ```
3. Open the app in Expo Go or a simulator.
